What are Objects?

Objects are collections of key-value pairs. They group related data and functionality together.

// Object literal
const person = {
    name: "Alice",
    age: 30,
    city: "New York"
};

console.log(person.name);  // "Alice"
console.log(person["age"]);  // 30

Properties & Methods

Adding Properties

const user = { name: "Bob" };

// Add new property
user.email = "bob@example.com";
user["phone"] = "555-1234";

console.log(user);
// { name: "Bob", email: "bob@example.com", phone: "555-1234" }

Methods (Functions in Objects)

const car = {
    brand: "Tesla",
    speed: 0,
    accelerate: function() {
        this.speed += 10;
    },
    brake: () => {
        this.speed -= 5;
    }
};

car.accelerate();
console.log(car.speed);  // 10

JSON (JavaScript Object Notation)

JSON is a text format for transmitting data. It looks like JavaScript objects but is language-independent.

Converting to/from JSON

// Object to JSON
const obj = { name: "Alice", age: 30 };
const json = JSON.stringify(obj);
console.log(json);
// {"name":"Alice","age":30}

// JSON to Object
const parsed = JSON.parse(json);
console.log(parsed.name);  // "Alice"

Real-World Example

// API Response (JSON)
const apiResponse = '{"id": 1, "title": "Learn JS", "done": false}';

// Parse it
const task = JSON.parse(apiResponse);
task.done = true;

// Send back
const updated = JSON.stringify(task);
console.log(updated);
