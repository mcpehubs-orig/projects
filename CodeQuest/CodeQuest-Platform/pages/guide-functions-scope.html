What are Functions?

Functions are reusable blocks of code that perform a specific task. Instead of writing the same code multiple times, you write it once in a function and call it whenever needed.

// Function definition
function greet(name) {
    console.log("Hello, " + name + "!");
}

// Function call
greet("Alice");  // Output: Hello, Alice!
greet("Bob");    // Output: Hello, Bob!

Benefits of Functions

  • DRY (Don't Repeat Yourself) principle
  • Better code organization
  • Easier debugging
  • Code reusability

Function Declaration Methods

1. Function Declaration

function add(a, b) {
    return a + b;
}

console.log(add(5, 3));  // 8

2. Function Expression

const subtract = function(a, b) {
    return a - b;
};

console.log(subtract(10, 4));  // 6

3. Arrow Function (ES6)

const multiply = (a, b) => a * b;

console.log(multiply(4, 5));  // 20

Parameters vs Arguments

Parameters: Variables in the function definition

Arguments: Actual values passed when calling the function

Understanding Scope

Scope determines where variables can be accessed in your code.

Global Scope

const globalVar = "I'm global";

function showVar() {
    console.log(globalVar);  // Accessible
}

showVar();  // Output: I'm global

Local Scope

function myFunction() {
    const localVar = "I'm local";
    console.log(localVar);  // Works
}

myFunction();  // Output: I'm local
console.log(localVar);  // Error! Not accessible

Block Scope (const/let)

if (true) {
    const blockVar = "I'm block scoped";
    console.log(blockVar);  // Works
}

console.log(blockVar);  // Error! Outside block
