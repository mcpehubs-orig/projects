Variables & Data Types

What are Variables?

Variables are containers that store values. Think of them as labeled boxes that hold information.

// Declaring variables
const name = "Alice";      // Text (String)
let age = 25;              // Number
var isStudent = true;      // Boolean (true/false)
let score;                 // Undefined (no value yet)

const vs let vs var

Keyword Scope Reassignable Use Case
const Block ❌ No Default choice (most safe)
let Block ✅ Yes When you need to change values
var Function ✅ Yes Legacy (avoid in modern code)

Operators

Arithmetic Operators

const a = 10;
const b = 3;

console.log(a + b);  // 13 (addition)
console.log(a - b);  // 7 (subtraction)
console.log(a * b);  // 30 (multiplication)
console.log(a / b);  // 3.33 (division)
console.log(a % b);  // 1 (modulo - remainder)
console.log(a ** b); // 1000 (exponentiation)

Comparison Operators

console.log(5 == "5");   // true (loose equality)
console.log(5 === "5");  // false (strict equality)
console.log(5 > 3);      // true
console.log(5 < 3);      // false
console.log(5 >= 5);     // true
console.log(5 <= 3);     // false
console.log(5 !== "5");  // true (not equal)

Logical Operators

const x = true;
const y = false;

console.log(x && y);  // false (AND - both must be true)
console.log(x || y);  // true (OR - at least one must be true)
console.log(!x);      // false (NOT - reverses boolean)
